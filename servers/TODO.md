# servers/ - MCP 서버 연동 설정

## 역할
이 프로젝트가 활용하는 MCP 서버들의 연동 설정과 래퍼를 관리합니다.

## 핵심 MCP 서버
1. **memorizer** - 지식 저장/검색/그래프 탐색
2. **pencil** - 블루프린트 시각화 (.pen 파일)

## 개발 예정
- [ ] MCP 서버 연동 설정 (.mcp.json)
- [ ] memorizer MCP 활용 패턴 문서화
- [ ] pencil MCP 활용 패턴 문서화
- [ ] MCP 호출 래퍼/유틸리티 (필요 시)

## 참고
- bkit servers/ (bkit-pdca + bkit-analysis) 패턴 참고
- harness는 자체 MCP 서버 개발보다 기존 MCP 활용에 집중
