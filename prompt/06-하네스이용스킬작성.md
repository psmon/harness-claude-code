이 프로젝트는 클로드 코드 호환 하네스를 지속적으로 디벨롭하는 프로젝트입니다.

클로드 코드 스킬 2.0 작성 가이드를 먼저 참고합니다.
- 웹검색 or 메모리검색 가능하며 최대한 최신 스킬가이드 작성을 먼저 이해

- 이 프로젝트의 하네스구조(md)를 참고(하위모두)

# 이 스킬의 목표
- 하네스 구조를 이용해 프로젝트를 생성및 수정하는데 이용
- harness-usage : 하네스를 이용해 프로젝트를 안전하게 생성및 수정하는 용도입니다.
  - harness-create : 생성부분은 이미 완성 별도로 작동됩니다. (참고) 
- 하네스 이용후 프로젝트 생성후.. 프로젝트검증을 포함,하네스가 잘 활용되었는지 평가항목도 추가하기 (아직 하네스 완성단계입니다.) - 지속발전용도

 # 이 스킬의 발동 트리거
- ex> 하네스를 이용해 프로젝트를 생성및 수정하려고합니다.
